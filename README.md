# 小テスト用プロジェクト・レポジトリ

## このレポジトリの概要

このレポジトリは、数回に渡って実施予定の小テスト用プロジェクトのレポジトリです。

各回の小テストの課題はこのレポジトリを通して公開されます。

公開された課題を各自のレポジトリに取り込んで、課された指示にしたがってプログラムを作成してください。

小テストをすべて完了すると、かんたんなレストランレビュサイトが完成します。
